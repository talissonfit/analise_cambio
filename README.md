# 📊 Análise de Volatilidade de Câmbio (USD/BRL) / Exchange Volatility Analysis

Este projeto utiliza **Python**, **NumPy** e **yfinance** para realizar uma análise estatística da variação do dólar frente ao Real brasileiro nos últimos 30 dias.

*This project uses **Python**, **NumPy**, and **yfinance** to perform a statistical analysis of the USD/BRL exchange rate variation over the last 30 days.*

---

## 🛠️ Tecnologias / Technologies
* **Python**
* **NumPy**: Processamento de arrays e estatística / *Array processing and statistics*.
* **yfinance**: Coleta de dados financeiros em tempo real / *Real-time financial data collection*.

## 📈 Funcionalidades / Features
- [x] Cálculo de Cotação Média, Máxima e Mínima / *Mean, Max, and Min rate calculation*.
- [x] Cálculo de Volatilidade (Desvio Padrão) / *Volatility calculation (Standard Deviation)*.
- [x] Identificação de dias de alta via Filtro Booleano / *Identifying "up" days via Boolean Filtering*.
- [x] Bypass de Rate Limit via Session Headers / *Rate Limit bypass using Session Headers*.

## 🚀 Como executar / How to run
1. Instale as dependências / *Install dependencies*: `pip install -r requirements.txt`
2. Execute o script / *Run the script*: `python analise_cambio.py`
